chaudio: programmatic music synthesis
=====================================

``chaudio`` is a python library for making music. Completely created through python source code, you can create virtually any music you can imagine.

This is the online user documentation for ``chaudio``. You can see all the source code for ``chaudio`` on ChemicalDevelopment's GitHub: `chaudio on GitHub <https://github.com/ChemicalDevelopment/chaudio>`_
